# Luna Flow

A beautiful, comprehensive mobile app that teaches women how to eat, move, and care for themselves in harmony with their menstrual cycle. Transform hormonal chaos into empowered wellness.

## Features

### Cycle Tracking
- Beautiful cycle wheel visualization showing current day and phase
- Automatic phase detection (Menstrual, Follicular, Ovulatory, Luteal)
- Educational hormone graph showing estrogen and progesterone curves
- Next period countdown
- Persistent data storage

### Nutrition Guide
- Phase-specific food recommendations with images
- Benefits explained for each food
- Tips for optimal eating during each phase
- Foods to avoid during each phase
- One-tap add to grocery list

### Movement Guide
- Cycle-synced workout recommendations
- Exercise intensity guidance based on current phase
- Duration and type suggestions
- Movement tips for each phase

### Self-Care Hub
- Phase-appropriate self-care activities
- Emotional support and validation
- Daily affirmations for each phase
- Journal prompts for reflection
- Understanding why you feel certain ways

### Smart Grocery List
- Add phase-specific recommended foods with one tap
- Custom item addition
- Organized by category
- Check off items as you shop
- Persistent storage

### Dark Mode
- Toggle between light and dark themes
- Perfect for nighttime tracking
- Beautiful deep purple dark theme
- Remembers your preference

## Tech Stack
- Expo SDK 53 / React Native 0.76.7
- TypeScript
- NativeWind (TailwindCSS for React Native)
- Zustand for state management
- React Native Reanimated for animations
- Expo Router for navigation
